
# Coding Test




## Run Locally

Clone the project

```bash
  git clone https://github.com/ruendyant01/fullstack-coding-test.git
```

Go to the project web directory

```bash
  cd 4
```

Frontend

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

Backend

Install dependencies

```bash
  npm install
```   
    
Set Env
->the env available on env template provided
->and the database connection

Start the server

```bash
  node app.js
```
